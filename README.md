## 프로젝트 소개

> React를 사용해 single page application으로 네이버 모바일 페이지를 클론 개발한 프로젝트입니다. 빌드엔 create-react-app을 사용하였고 json-server를 통해 가상의 서버를 만들어 회원가입한 유저값과 웹툰 데이터를 저장하였습니다.
